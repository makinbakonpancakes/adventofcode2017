# Advent Of Code 2017
To execute any of the solultions go to the appropriate directory and
use the following commands depending on the language.

## Python
```shell
python dayXX.py
```

## Java
```shell
javac DayXX.java
java -classpath . DayXX
```

## Racket
```shell
racket dayXX.rkt
```
## C++
```shell
g++ dayXX.cpp
./a.out
```

## Javascript
Just open the `.html` file in a browser

## Clojure
```shell
clojure dayXX.clj
```

## Ruby
```shell
ruby dayXX.rb
```
